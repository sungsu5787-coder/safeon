# 재발방지대책 참고자료 — 산업안전보건기준에 관한 규칙

> 근거. 「산업안전보건기준에 관한 규칙」(약칭 안전보건규칙) 고용노동부령 제417호, 시행 2025. 6. 29.
> 용도. 사고보고서·안전보고서의 **재발방지대책** 작성 시 사고 유형에 맞는 법적 조치사항을 빠르게 찾기 위한 참고표.
> 사용법. 사고 유형(아래 목차)을 찾아 해당 조항의 핵심 조치를 재발방지대책 문구로 활용한다. 인용 시 `제N조` 형식으로 표기한다.

## 1. 추락

높이 또는 깊이 2미터 기준이 핵심이다.

| 조항 | 핵심 조치 |
|------|-----------|
| 제42조 | 작업발판 설치, 곤란 시 추락방호망(작업면에서 10m 이내, 처짐 짧은 변의 12% 이상) 설치, 그것도 곤란하면 안전대 착용 |
| 제43조 | 개구부·작업발판 끝에 안전난간·울타리·덮개 등 방호조치, 어두운 곳은 개구부 표시 |
| 제44조 | 2m 이상 장소에서 안전대 부착설비 설치, 작업 시작 전 부착설비 점검 |
| 제13조 | 안전난간 구조(상부난간대 90cm 이상, 발끝막이판 10cm 이상, 100kg 하중 견딜 것) |
| 제45조 | 지붕 위 작업 시 가장자리 안전난간, 채광창 덮개, 슬레이트 위 30cm 이상 발판 |
| 제46조 | 높이·깊이 2m 초과 시 승강설비(건설용 리프트 등) 설치 |
| 제32조 | 추락 위험작업에 안전모, 2m 이상 추락위험 장소에 안전대 지급·착용 |

**재발방지 문구 예시.** 작업발판 끝·개구부에 안전난간을 견고하게 설치하고, 설치가 곤란한 구간은 추락방호망 또는 안전대 부착설비를 갖춰 안전대를 착용하도록 한다(제42조·제43조·제44조).

## 부록 — 재발방지대책 작성 체크리스트

사고보고서 작성 시 아래 4개 축으로 대책을 구성하면 누락이 적다.

1. **공학적 대책(설비).** 방호장치·국소배기·안전난간·방호망 등 물리적 설비 보완. (해당 조항 인용)
2. **관리적 대책(절차).** 작업계획서·작업허가서·LOTO·신호체계·출입통제·점검주기. (제38조·제92조·제40조)
3. **교육적 대책(사람).** 특별안전교육·TBM·유해성 주지·자격자 배치. (제35조·유해성 주지 조항)
4. **보호구(개인).** 안전모·안전대·호흡용 보호구·청력보호구 등 지급·착용 점검. (제32조)

> 작성 원칙. "~한다"로 끝맺고 근거 조항을 괄호로 병기한다. 추상적 표현("주의한다") 대신 구체적 설비·수치·주기를 명시한다.
